¡Pon en práctica lo que has aprendido!

Objetivo:

El objetivo de este ejercicio es poner en práctica los conocimientos que has adquirido en las clases previas.

Resuelve los siguientes ejercicios:

  • Crea un encabezado con el título de la página y añade secciones. Añade párrafos donde lo necesites para explicar el contenido de las secciones.
  • En la primera sección crea una lista ordenada con un ranking de tus series, pelis, libros o videojuegos favoritos.
  • En la segunda sección crea una lista desordenada sobre tus aficiones.
  • En la tercera sección añade el siguiente contenido multimedia:

    • Una de estas imágenes o las 3, con un ancho y alto de 400px:
      • https://i.imgflip.com/3y3hxm.jpg
      • https://miro.medium.com/v2/resize:fit:1400/format:webp/1*tBkBBEnKeOGo7eUaMW2GxA.jpeg
      • https://i.imgflip.com/u9pv5.jpg

    • Este gif con 400px de ancho y alto: https://media.giphy.com/media/MeJgB3yMMwIaHmKD4z/giphy.gif

  • Crea un header con una barra de navegación con el nombre de las secciones.

    BONUSIntenta crear un enlace que lleve a cada una de las secciones. 🔎 Pista: IDs y "anclas"

  • Un formulario en otra sección, o si te atreves en otra página, sobre lo que quieras dividido por secciones con los siguientes elementos:

    • Fieldset
    • Legend
    • Label
    • Input tipo email
    • Input tipo text
    • Input tipo checkbox
    • Select
    • Textarea
    • Botones para enviar y borrar todo

    • Si hay algún elemento que no conoces puedes buscarlo en internet

  • Footer con enlaces a páginas de interés, como Linkedin u otras redes sociales
